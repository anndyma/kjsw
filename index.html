⚠️ 无法在本地直接双击运行

检测到您正在直接双击打开 HTML 文件（浏览器地址栏为 file:///）。

因为本系统采用了前沿的模块化加载架构，现代浏览器处于最高安全保护级别，严禁跨文件读取和写入本地数据库，这会导致建账和模块加载功能受限或断电卡死！

🚀 极速部署方案（2选1即可）：

  • 一键生成云端网页（推荐）：将装有所有文件的整个文件夹，用鼠标拖拽到 Netlify Drop 官方网站 的圆圈中，5 秒内即可免费生成手机电脑皆可访问的公网链接！
  • 本地开发运行：使用 VS Code 编辑器打开本文件夹，安装 Live Server 插件，右键 index.html 启动运行。
